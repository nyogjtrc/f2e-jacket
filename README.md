# f2e-jacket
f2e project sample

## Quick Start

```
$ git clone git@github.com:nyogjtrc/f2e-jacket.git
$ cd f2e-jacket/
$ yarn
$ yarn start
```

## Library

- react
- react-dom
- react-router-dom

## Development

Run `npm start` or `yarn start` to start develop server
Open http://localhost:8080/ to view it

Run `npm run build` or `yarn build` to build app
